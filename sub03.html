Project Workflow

01

Discovery

  • 제약회사 웹사이트 레퍼런스 분석
  • 기업소개, 허가제품, ESG 중심의 페이지 구조 기획
02

Design

  • 신뢰성과 전문성을 표현하는 네이비 블루 계열 컬러 팔레트 선정
  • Whisk AI로 생성한 실험실/연구 현장 이미지 활용
  • Figma로 와이어프레임 및 UI 디자인
03

Development

  • fetch API로 header/footer를 별도 HTML 파일로 분리해 모든 페이지에서 공통 로드
  • IntersectionObserver로 스크롤 진입 시 섹션 페이드인 애니메이션 구현
  • IntersectionObserver로 카드 순차 등장(0.1초 간격) 애니메이션 구현
  • 숫자 카운터 애니메이션 구현 (0 → 100, setInterval 기반)
  • 시맨틱 table로 허가제품 정보 구조화
  • Google Maps iframe 삽입, loading="lazy"로 초기 로딩 최적화
04

QA

  • 크로스 브라우징 테스트 (Chrome, Safari)
  • 반응형 검증
05

Deploy

  • GitHub Pages 배포

Design Decisions

주요 디자인 결정과 그 이유

왜 블루 컬러인가?

#032753의 네이비 블루로 신뢰감과 권위 있는 느낌을 주고, #ACBFFF의 밝은 블루로 포인트를 주어 딱딱하지 않은 균형감을 만들었습니다. 의료/제약 분야에서 신뢰성을 전달하는 데 가장 적합한 컬러 조합입니다.

Whisk AI 이미지 활용

실제 실험실, 연구시설, 제조 현장 이미지를 Whisk AI로 생성해 활용했습니다. 제약회사의 R&D 역량과 전문성을 시각적으로 전달하기 위한 선택입니다.

Header/Footer 분리

fetch API로 header/footer를 별도 HTML 파일로 분리해 모든 페이지에서 공통으로 불러오도록 구현했습니다. 메뉴 수정 시 한 파일만 수정하면 전체 페이지에 반영됩니다.

스크롤 애니메이션

IntersectionObserver로 스크롤 진입 시점에만 애니메이션이 실행되도록 구현했습니다. 카드는 0.1초 간격으로 순차 등장하고, 섹션은 페이드인 처리해 자연스러운 흐름을 만들었습니다.

Visual Section

기업 슬로건과 CTA 버튼을 배치했습니다. 배경 이미지는 Whisk AI로 생성한 연구 현장 이미지를 사용했습니다.

Section 1

3개의 핵심 사업 영역을 카드 형태로 제시했습니다. IntersectionObserver로 스크롤 진입 시 카드가 순차적으로 등장하고, hover 시 이미지 확대 효과를 CSS transition으로 구현했습니다.

Section 3

뉴스/공지사항을 2컬럼으로 구분하고, 전체 배경을 이미지로 처리해 섹션을 시각적으로 구분했습니다. hover 시 해당 기사가 강조되도록 CSS 효과를 적용했습니다.

Section 4

Google Maps를 iframe으로 삽입하고 loading="lazy"를 적용해 해당 섹션 진입 시점에 지도가 로드되도록 했습니다.

Header & Footer

fetch API로 header/footer를 별도 파일로 분리해 모든 페이지에서 공통 로드되도록 구현했습니다. 현재 페이지에 맞는 GNB 메뉴가 자동으로 활성화됩니다.

All Sections

Mission, Vision을 각각 풀스크린 이미지 위 텍스트 오버레이로 구성했습니다. 스크롤 시 IntersectionObserver로 해당 섹션 진입을 감지해 텍스트 박스가 페이드인되도록 구현했습니다.

Statistics Section

"100여 개국"의 글로벌 허가 실적을 숫자 카운터 애니메이션으로 표현했습니다. IntersectionObserver로 섹션 진입 시점에 setInterval 기반 카운터(0 → 100)가 실행되고, 한 번만 동작하도록 animated 클래스로 중복 실행을 방지했습니다.

Product Visual

바이오의약품, 케미컬의약품, 제품소식 카테고리를 상단 메뉴로 배치하고, 현재 활성화된 메뉴에 .on 클래스를 적용해 시각적으로 구분했습니다.

Product Table

시맨틱 table로 제품명, 이미지, 성분명, 적응증, 글로벌 허가기관 정보를 구조화했습니다. 바이오시밀러/바이오신약은 탭 버튼으로 구분했습니다.

Disclaimer

의료 정보용으로만 사용되며 홍보/판매 목적이 아님을 하단에 명시했습니다. 의료 관련 웹사이트에서 필수적으로 포함해야 하는 법적 고지사항입니다.

ESG Menu

지속가능 경영 보고서, ESG 평가등급, 인증서 카테고리를 상단 메뉴로 배치하고, 현재 활성화된 메뉴에 .on 클래스를 적용해 시각적으로 구분했습니다.

ESG Report Cards

연도별 ESG 보고서를 카드 형태로 나열했습니다. IntersectionObserver로 카드가 화면에 진입할 때 0.1초 간격으로 순차 등장하도록 구현하고, hover 시 이미지 확대 효과를 CSS transition으로 적용했습니다.
